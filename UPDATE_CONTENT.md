# AutoDoor 更新内容

## 文档说明

此文件用于存储每次版本更新的内容说明，GitHub Actions 在构建时会读取此文件的内容并用于创建 GitHub Release 的更新说明。

## 更新内容格式

每次更新时，请在文件末尾添加新的版本更新内容，格式如下：

```markdown

### 更新内容
- 实现版本自动检查功能

### 优化内容
- 在程序开始运行和结束运行时播放音频

### 修复内容
- 无

```

## 示例

```markdown
## [1.5.0] - 2024-01-29

### 更新内容
- 新增版本更新提醒功能
- 支持自动检查版本更新
- 支持手动检查版本更新
- 发现新版本时显示美观的更新通知
- 使用飞书文档链接作为更新跳转地址

### 修复内容
- 修复打包配置中的依赖排除问题
- 确保 requests 及其依赖被正确包含在打包文件中

### 改进内容
- 优化版本比较算法
- 完善错误处理和日志记录
```

## 当前更新内容

### 自动构建版本信息
